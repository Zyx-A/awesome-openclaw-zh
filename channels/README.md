# 渠道接入中心

如果你已经把 OpenClaw 部署好了，下一步就从这里开始，把它接到真实使用的聊天和协同入口。

本目录主要解决两件事：

- 把 OpenClaw 接到国内常用渠道
- 明确每条渠道的能力边界、接入方式和验证步骤

## 当前可用

- 企业微信：看 [企业微信智能机器人接入](./01-wecom-intelligent-bot.md)

## 目录约定

后续这里会逐步拆成两类：

- 原生渠道：OpenClaw 官方已经支持的渠道
- 桥接渠道：通过平台机器人、Webhook 或自建适配层接入的渠道

## 建议阅读顺序

1. 先完成 [部署与安装中心](../deploy/README.md)
2. 再选择 1 个渠道接入
3. 跑通第一个真实对话或推送
4. 回到 [5分钟快速上手](../quickstart/00-5min-quickstart.md)
